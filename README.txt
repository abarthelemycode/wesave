L'objectif de l'exercice est d'intégérer la maquette 'maquette.jpg'.
Il s'agit d'une mini appli avec deux onglets 'Mes albums' et 'Mes playlists'.
Le contenu des deux listes doit être rempli à partir du fichier 'src/data.js'
(équivalent au retour d'un appel à un web service).

Le site doit être compatible IE >= 10
Pas de limite de temps mais préciser lors du rendu le temps passé (environ).
Organisation du code totalement libre.

Sont autorisés :
- React et Vue, mais il n'est pas obligatoire de les utiliser
- les préprocesseurs CSS type Less, Sass
- les outils de build (Grunt, Gulp, Webpack, etc.)

Ne sont pas autorisés :
- Bootstrap et autres libs CSS
- les libs JS type JQuery, Zepto, Underscore, Lodash, etc.

La police utilisée est Lato (disponible sur Google Font).
